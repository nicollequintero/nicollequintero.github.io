Enumerating through Fitness Bootcamp

July 20, 2014

Enumerable methods are built-in features of Ruby that allow us to manipulate data easily. One way to think about it is that someone thought of all the common things you'd want to do with data - sort it, find the largest value, find the smallest value, etc. and did 99% of the work for you. All that's left is to call that method into action when you need it.



Enumerable#map

The map method looks through your list of data (for example, string, hash, array) and performs a block of code on every single item.

Let's enroll in Fitness Bootcamp! You are a gym that has a list (in this case, an array) of people who want to sign up for your classes.

The map method allows you to do something to each of those elements in your prospects. In this case, let's go ahead and print a statement that each prospect is now enrolled. Map will execute the block of code for every element of that array.

RESULT

Enumerable#group_by

Now that we have our members enrolled, we also keep track of their fitness progress. In this instance, we've recorded how many situps each person can do in a minute. We've stored this data into a hash.

The group_by method allows you to group the elements in your hash by a criteria of your choosing. In our case, we want to separate those who can do 40 or more pushups in a minute, from those who cannot. We'll end up with two groups - advanced and beginner. These new terms "Advanced" and "Beginner" will be the keys in a new hash. The values to these keys will be the name and number of situps of those who fall into each category.

RESULT

Enumerable#cycle

Let's say you have a series of exercises you want each member of Fitness Bootcamp do to. In order to give clear directions, you'll want to print out detailed directions reflecting how many times they are to do each exercise.

In this case, we're representing the steps to do a burpee in an array.

The cycle method allows you to execute a block of code for each element in your array any number of times you desire. If I want my members to do 10 burpees, then I'll cycle through the array 10 times and print each step along the way.
